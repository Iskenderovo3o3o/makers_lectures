git init превращает текущую директорию в которой вы находитесь в git репозиторий 

git add название файла /. - отслеживание указанных/всех файлов 

git commit -m "message for commit" - создание версии кода с сообщением 
 
git remote <name_of_remote> <ssh_key> -  связывание локального репозитория с удаленным

git push  <name_of_remote> <branch_name> - оправка версии на гитхаб
